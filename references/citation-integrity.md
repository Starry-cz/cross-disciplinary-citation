# 引用完整性审计

## 每条记录

- [ ] 作品存在且链接、DOI、报告号、标准号、专利号或 accession number 可核验。
- [ ] 作者/机构、题名、年份、版本、出处和位置与原始记录一致。
- [ ] 使用的是实际阅读且与论断对应的版本；预印本、正式版、更正和撤稿状态未被混淆。
- [ ] 文献类型与载体相符：数据、软件、代码、标准、报告和网页没有伪装为期刊文章。

## 每处引文

- [ ] 引文紧邻其支持的事实、方法、定义、数据或直接引语；引语有页码、段落号或时间戳（若适用）。
- [ ] 文内形式与文后条目或脚注唯一对应；多文献并列不是为了堆砌权威。
- [ ] 转引、个人通信、AI 协助和不可公开材料被透明说明，并符合目标规范。
- [ ] 顺序编号、作者—年份、脚注和法学引证没有未经允许地混合。

## 交付前

- [ ] 使用目标期刊/学校最新指南或官方样式文件复核特殊类型。
- [ ] 更新 Word 域，检查可点击跳转、编号、上标/字体和书签完整性。
- [ ] 运行本技能校验脚本，并保留源文档与映射文件备份。
