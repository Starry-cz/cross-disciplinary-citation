# 常用文内引文模式

以投稿期刊的《作者须知》为最终准则；以下为中文学术论文中常见且可用于映射文件的形式。

| 情形 | 文内写法 | 映射键示例 |
| --- | --- | --- |
| 单一中文作者 | （滕洋，2023） | `滕洋，2023` |
| 多位中文作者 | （李木洲、孙艺源，2024） | `李木洲、孙艺源，2024` |
| 三人以上中文作者 | （景安磊等，2024） | `景安磊等，2024` |
| 叙述式中文 | 景安磊等（2024）指出 | `景安磊等` 或 `2024` 均不应单独建链接；用完整括注式改写后再处理 |
| 单一英文作者 | (Smith, 2024) | `Smith, 2024` |
| 两位英文作者 | (Wang & Li, 2024) | `Wang & Li, 2024` |
| 三人以上英文作者 | (Smith et al., 2024) | `Smith et al., 2024` |
| 多篇并列 | （滕洋，2023；景安磊等，2024） | 每项各一行，保留分号为普通文本 |
| 同作者同年 | （王明，2024a，2024b） | `王明，2024a`、`2024b` 必须均可唯一映射 |
| 附页码 | （王明，2024：15） | 如期刊允许，应把页码作为引文可见文本的一部分，例如 `王明，2024：15` |
| 机构作者 | （教育部，2025） | `教育部，2025` |
| 无日期 | （教育部，n.d.） | `教育部，n.d.`，并在文后按期刊要求标注日期缺失 |
| 顺序编码 | [1]、[1-3]、[1,3] | 采用 `ref-citation-embedder`，不要与著者—年份脚本混用 |

## 转引

优先查阅原始文献。确有必要时，按期刊规范写“作者（年份，转引自作者，年份）”或同义形式；文后通常只著录实际阅读的文献。不要把未阅读的原始出处伪装为已直接引用。

## 交叉引用的可链接范围

脚本将映射键本身制作为一个可点击 REF 域。外层括号、分号、顿号和连接词保持为普通文字。因此，多篇并列引文应把每篇拆为单独的映射键，而不是将整串引文映射到一个编号。

## GB/T 7714 文后条目提示

文后条目须保留 `[M]`、`[J]`、`[D]`、`[EB/OL]` 等文献类型标识，并以顺序编号 `[N]` 开头，供 Word 书签定位。著者—年份制影响的是正文显示方式，不取消文后条目的著录规范。
