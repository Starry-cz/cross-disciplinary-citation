# 引文体系与学科路由

先读目标期刊、会议、学位论文模板或课程要求；下表只用于缺少特殊规定时提出可验证的默认方案。

| 目标或惯例 | 常见体系 | 优先检查 |
| --- | --- | --- |
| 中国通用论文、部分社科与工程期刊 | GB/T 7714 顺序编码或著者—出版年 | 期刊是否指定两种体系之一、文献类型标识与中文标点 |
| 心理、教育、管理、传播与行为科学 | APA 7 作者—年份 | 作者数量、DOI/URL、数据与软件条目 |
| 历史、文学、艺术、宗教与档案研究 | Chicago Notes-Bibliography 或 MLA | 页码、档案馆/馆藏、首次注释与缩略注释 |
| 经济、政治、地理与部分科学 | Chicago Author-Date、APA 或 CSE Name-Year | 年份区分、数据与工作论文版本 |
| 计算机、电子、通信、控制与工程 | IEEE 顺序编码 | 会议录、标准、专利、在线先行与 DOI |
| 医学、生命科学与临床 | Vancouver/NLM、AMA 或期刊自定义 | 作者截断规则、期刊缩写、试验注册号、指南/数据 |
| 化学与材料 | ACS 或期刊自定义 | 作者—年份/编号/上标变体、专利与谱图/数据 |
| 生物、生态、农业 | CSE 或期刊自定义 | CSE 的 Name-Year、Citation-Sequence、Citation-Name 三种体系 |
| 法律与政策 | 适用法域规范 | 判例、法条、版本、法院/法域、段落号 |

## 选择规则

1. 有投稿指南时，完全采用其指定版本，即使它与学科惯例不同。
2. 有期刊 CSL、EndNote/Zotero 样式或官方模板时，以其输出为基准，再人工核对特殊来源。
3. 未指定时，先向用户说明拟用体系；不要自行把一篇已有编号制稿件改为作者—年份制。
4. 同一文稿只保留一个主体系。补充材料、脚注和法规引注仅在规范明确允许时例外。
